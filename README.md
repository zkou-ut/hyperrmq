# hyperrmq

This repository contains implementations of average-case optimal RMQ data structures for random permutations and arrays with increasing runs.

## Setup

```bash
bash setup.sh
cd pizza\&chilli
bash download.sh
```

## Test

```bash
./build/simple-test
./build/test/hyperrmq-test
```
